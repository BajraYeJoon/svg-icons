# Getting Started

Welcome to our SVG Icon Library! This guide will help you get started with using our icons in your projects.

## Installation

```bash
npm install @example/svg-icons
```

## Basic Usage

```jsx
import { ArrowRight } from '@example/svg-icons';

function MyComponent() {
  return <ArrowRight className="w-6 h-6" />;
}
```

## Customization

All icons support Tailwind CSS classes for easy customization:

```jsx
<ArrowRight className="w-6 h-6 text-blue-500 hover:text-blue-600" />
```
